golibs of ocean(me)


FOR GITLAB:
```
	get libs:
	go get gitlab.com/oucaijun/oceangolibs

	update libs:
	go get -u gitlab.com/oucaijun/oceangolibs
	
		
	use:
	import "gitlab.com/oucaijun/oceangolibs/hello"
```
